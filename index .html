Rules of the game

This is a simple maze game where you have to find the path from the source to the destination. The maze is generated randomly and you can choose the size of the maze. You can also see the solution of the maze by clicking on the "Show Solution" button.

Start at the source(0,0) with an empty string and try every possible path i.e upwards(U), downwards(D), leftwards(L) and rightwards(R)
